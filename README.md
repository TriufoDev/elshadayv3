# Elshaday Poços Artesianos

## 📋 Descrição

Site institucional da empresa Elshaday Poços Artesianos, especializada em perfuração e construção de poços artesianos e semi-artesianos. A empresa oferece serviços completos relacionados à captação de água subterrânea com qualidade e profissionalismo.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica do site
- **CSS3**: Estilização responsiva com animações
- **JavaScript**: Interatividade e animações de scroll
- **Font Awesome**: Ícones vetoriais
- **Google Fonts**: Tipografia Poppins

## ✨ Funcionalidades

- **Design Responsivo**: Totalmente adaptável para desktop, tablet e mobile
- **Animações de Scroll**: Elementos aparecem suavemente ao rolar a página
- **Header Fixo**: Navegação sempre visível no topo
- **Botões Interativos**: Animações e efeitos hover
- **WhatsApp Integration**: Links diretos para contato via WhatsApp
- **Mapa Integrado**: Localização da empresa via Google Maps
